Welcome to your COM2042 group repository. Please feel free to change this README to tell us more about your project.

You may want to consider using the issue tracker, CI/CD jobs (incl pipeline badges), webhooks, runners, wiki, merge requests, and even pages (for static sites), which are all provided for free as part of our GitLab instance. CI/CD jobs could include running automated tests on commits, as part of merge requests for QA, generating docs from docstrings, generating artefacts for your deployment server to pull in, calling webhooks (network firewalls permitting) etc. We'll provide some resources to help you get started as the project progresses.

How you use this repo is up to your team, but you must use this for your project's code.

We will make changes to your repo when the deployment environments are setup. For now, you will have your local development environment... Soon you will also have a production/staging environment!

Not everything is enabled by default in your repo. If you are having issues or want a feature enabling, please let your sponsor know who can submit a request for you (e.g., webhook configs).
